# VC CRM UI Execution Pack

This pack is for improving the UI/UX of an already-developed 12-phase VC CRM system.

Goal:
- Keep existing backend and module logic.
- Replace cluttered screens with a clean, white, modern CRM interface.
- Use a consistent AppShell, page templates, tables, forms, cards, dashboards, and detail views.
- Make the system easy for founders, sales, delivery, HR, finance, admin, and super admin users.

Use this pack with Codex after the first 12 development phases are complete.
